# Quinect

**Quinect**는 온라인 실시간 교육용 퀴즈 플랫폼입니다.  
실시간 상호작용(퀴즈, 채팅, 투표, 토론)과 데이터 기반 분석을 통해  
학습자와 강사가 함께 참여하는 몰입형 학습 환경을 제공하는 웹 서비스입니다.

---

## 주요 기능

### 인증 / 계정 시스템

- 이메일 기반 회원가입 및 로그인
- Google / Kakao OAuth2 소셜 로그인
- 비밀번호 찾기 및 재설정
- 세션 만료 및 자동 로그아웃 처리
- 회원 탈퇴(비활성 → 일정 기간 후 영구 삭제)
- 프로필 편집 (닉네임, 소개, 프로필 이미지)
- 활동 타임라인(푼 문제, 업적, 포인트 이력 등) 조회

### 포인트·보상 / 업적

- 퀴즈 풀이, 실시간 퀴즈 참여, 출석 체크 등 활동에 따른 포인트 지급
- 포인트 상점에서 프로필 테두리, 테마 등 꾸미기 아이템 구매
- 포인트 획득 / 사용 내역 기록
- 업적 시스템 (조건 달성 시 뱃지/업적 해금, 해금/미해금 상태 확인)
- 출석부 페이지 (출석한 날짜 시각화)

### 퀴즈 콘텐츠

- **비동기 퀴즈 풀기**
  - 퀴즈 리스트에서 선택 → 문제 풀이 → 자동 채점 → 포인트 지급
- **문제 출제**
  - 문제 유형: 객관식 / 주관식(서술형)
  - 문제/선지 추가·삭제, 정답 및 배점 설정
  - 점수 공개/비공개 옵션 (결과 즉시 공개 여부 제어)
  - 임시 저장 및 나중에 불러오기
- **AI 기반 퀴즈 생성**
  - 주제 / 난이도 / 문제 수 / 유형을 지정하면 Gemini API로 자동 퀴즈 생성
  - 자동 생성된 문제를 편집 후 저장 가능

### 실시간 방 & 퀴즈 세션

- 방 생성(호스트) / 방 입장(게스트, 회원)
- 고유 방 코드 + QR 코드로 참여 링크 공유
- 대기실에서 준비 상태 / 참여 인원 표시
- 실시간 퀴즈 진행
  - 모든 참가자 제출 시 다음 문제로 진행
  - 현재 점수·순위를 실시간으로 보여주는 랭킹 영역
  - 남은 문제 수, 현재 문제 정보 표시
- 최종 결과 페이지
  - 1~3등 강조, 전체 순위 리스트
  - 점수에 비례한 포인트 지급(회원만)

### 실시간 상호작용 / 커뮤니티

- **실시간 채팅**
  - 방 내 참가자 간 메시지 송수신
  - 시간/발신자/스타일 구분으로 가독성 확보
- **투표 / 찬반 토론 (방장 전용)**
  - 간단한 투표 생성 → 참가자 찬성/반대 집계
- **친구 및 초대**
  - 이메일 기반 친구 추가 / 수락 / 거절 / 차단
  - 친구 목록 / 검색 / 관리
  - 친구 DM(1:1 채팅)
  - 방 초대(채팅/알림을 통해 방 참가 유도)
- **알림 시스템**
  - DM, 업적 달성, 친구 초대 등 이벤트를 실시간 알림으로 표시
  - 알림 클릭 시 해당 기능/페이지로 바로 이동

## 사용자 플로우

### 게스트(비회원)

1. 메인 페이지 접속  
2. “방 입장”에서 방 코드 입력  
3. 닉네임·캐릭터 선택 후 실시간 퀴즈 방 입장  
4. 채팅/투표/퀴즈 참여 (포인트/기록은 적립되지 않음)

### 회원

1. 로그인 또는 소셜 로그인  
2. 메인 대시보드 진입
3. 기능 선택
   - 마이페이지: 프로필, 출석, 업적, 타임라인
   - 퀴즈: 퀴즈 풀기, 퀴즈 출제, AI 퀴즈 생성
   - 실시간: 방 생성(호스트), 방 입장(게스트)
   - 상점: 포인트 상점, 아이템 구매
4. 활동 결과는 포인트·업적·로그로 축적

## 데이터 모델 개요 (ERD 요약)

> 실제 ERD는 `docs/erd.png` 와 같이 이미지로 함께 관리하는 것을 권장합니다.

- **사용자 영역**
  - `user`, `user_profile`, `friendship`, `friend_message`
  - `user_activity_log`, `user_activity_point_change`, `user_activity_attendance`
  - `achievement`, `user_achievement`
- **경제/아이템 영역**
  - `shop_item`, `user_inventory`
- **실시간 방 / 세션 영역**
  - `room`, `participant`, `room_quiz`
- **퀴즈 영역**
  - `quiz`, `quiz_question`, `quiz_option`
  - `quiz_submission`, `quiz_answer`, `quiz_grading`
- **알림 / 공통**
  - `notification`, `notification_recipient`
  - `board_post`, `code_table`

## 기술 스택

### Frontend

- **HTML / Thymeleaf**
- **JavaScript / jQuery**
- **CSS3 / Bootstrap**

### Backend

- **Java 21 (JDK 21)**
- **Spring Boot 3.5.7**
- Spring MVC / Spring Data JPA / Spring Security
- WebSocket(STOMP) 기반 실시간 통신

### Database

- **MySQL 8.0.43**

### External & Infra

- Gemini API (AI 퀴즈 자동 생성)
- Zebra (QR 코드 생성)
- OAuth 2.0 (Google / Kakao)
- 기타: Lombok, Validation 등 Spring 생태계 라이브러리
